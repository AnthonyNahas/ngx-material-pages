 Library's components This library consists of 4 essential components.

ngx-material-pages the main container that hold the pages incl. the outlook view and the content

ngx-material-page-loader the container of each page. This component should have two nested elements which are the outlook and the content components (see below)

ngx-material-page-outlook the outlook view of a page. This can contain a custom component, custom layout or just a title.

ngx-material-page-content this is the content view of the page. This can be anything!!
